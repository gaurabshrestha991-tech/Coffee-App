# Coffee-App
This is a Python-based Coffee Shop Management System that allows users to browse a coffee menu, place orders, and manage their current order.
A simple Coffee Shop Management System built using Python.

## Features
Display a coffee menu with prices
Add coffee items to an order
View the current order
Calculate the total bill
Beginner-friendly and easy to understand

## Technologies Used
Python 3
Object-Oriented Programming (OOP)

## Project Status
This project is currently under development. More features such as removing items, saving orders, and improved user interaction will be added in future updates
